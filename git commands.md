### git clone

Скачать репозиторий к себе на компьютер

> Скопируй мне эту папку с GitHub на мой компьютер

`git clone https://github.com/user/project.git`

---

### git pull

Забрать свежие изменения с GitHub

> Обнови мою локальную папку тем, что сейчас лежит на GitHub

`git pull`

---

### git add

Подготовить файлы к сохранению

> Вот эти файлы я хочу сохранить в истории

`git add testcases.md     # один файл`

`git add .                # все изменения сразу`

---

### git commit

Сохранить изменения с комментарием

> Сохрани изменения и подпиши, что я сделал

`git commit -m "Добавил тест-кейсы для авторизации"`

---

### git push

Отправить свои коммиты на GitHub

> Загрузи мои сохранённые изменения в облако (GitHub)

`git push`

---

### git merge

Объединить изменения

> Соедини изменения из одной ветки с другой

`git merge feature-branch`
